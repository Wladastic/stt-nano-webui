STT WebUI API

OpenAI-compatible Speech-to-Text HTTP API. Models are loaded on-demand and unloaded after an idle TTL (…s) to free VRAM. Multiple models can be cached simultaneously if VRAM permits.

Base URL: http://<host>:7861 (this page: )

Auth: none. CORS: open. Audio formats: wav, mp3, flac, ogg, m4a, webm (anything ffmpeg can decode). Internally resampled to 16 kHz mono.

Available Models

POST /v1/audio/transcriptions

Transcribe an audio file. OpenAI-compatible (drop-in for openai.audio.transcriptions.create). multipart/form-data.

FieldTypeRequiredDescription
filefileyesAudio file upload
modelstringnoModel id (see list above). Default whisper-turbo. Alias whisper-1 maps to whisper-turbo.
languagestringnoISO code hint (en, de, …). Whisper auto-detects if omitted; Parakeet ignores this.
response_formatstringnoOne of json (default), verbose_json, text, srt, vtt.

Response — json

{ "text": "Hello world." }

Response — verbose_json

{
  "text": "Hello world.",
  "language": "en",
  "duration": 2.34,
  "segments": [
    { "id": 0, "start": 0.0, "end": 2.34, "text": "Hello world." }
  ]
}

Response — text / srt / vtt

Plain text body (text/plain). srt/vtt are subtitle formats with timestamped segments.

Example — curl

curl -X POST http://<host>:7861/v1/audio/transcriptions \
  -F "file=@audio.wav" \
  -F "model=parakeet-onnx-int8" \
  -F "response_format=verbose_json"

Example — Python (OpenAI SDK)

from openai import OpenAI
client = OpenAI(base_url="http://<host>:7861/v1", api_key="not-needed")
with open("audio.wav", "rb") as f:
    r = client.audio.transcriptions.create(model="whisper-turbo", file=f)
print(r.text)

GET /v1/models

OpenAI-compatible model listing. Returns {"object":"list","data":[{"id","object","owned_by","description"}, …]}.

GET /v1/status

Per-model load state, idle/TTL countdown, and VRAM usage. Useful for observability and pre-warming decisions.

{
  "ttl_seconds": 300,
  "models": {
    "parakeet-onnx-int8": { "status": "loaded", "idle_seconds": 12, "ttl_remaining": 288, "ttl_total": 300 },
    "whisper-turbo": { "status": "unloaded" }
  },
  "vram": { "total_mb": 24564, "allocated_mb": 432, "reserved_mb": 512 }
}

POST /v1/models/{name}/load

Pre-load a model into VRAM (blocking until ready). Use to avoid first-request latency.

POST /v1/models/{name}/unload

Unload a single model and free its VRAM.

POST /v1/models/flush

Hard reset: unload all models, run GC, empty the CUDA cache, and IPC-collect. Use when VRAM looks stuck.

Error format

Standard FastAPI errors: {"detail": "..."} with HTTP 4xx/5xx. Unknown model → 400. Internal failure → 500.

Notes for agents

  • First request to an unloaded model blocks on load (Parakeet NeMo: ~10s, ONNX: ~5s, Whisper: ~5s on warm cache, longer on cold HF download).
  • If you plan back-to-back requests, hit /v1/models/{name}/load first; subsequent /v1/audio/transcriptions calls reuse the cached model and reset its TTL.
  • Parakeet models are English-leaning and ignore the language field; pick Whisper for non-English or auto-detect.
  • parakeet-onnx-int8 is the lightest (~432 MB VRAM); parakeet (NeMo) is heaviest but most accurate; whisper-turbo is the multilingual choice.
